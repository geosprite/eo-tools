# eo-tools-snap

SNAP tools for Earth Observation Tools.

This package provides Sentinel-1 SNAP preprocessing tools under
`geosprite.eo.tools.snap`. It depends on `eo-tools-core` for the shared tool
protocol and registry helpers, and `eo-stac` from `../eo-libs/stac` for asset
models.

When installed, the package is discovered through the `geosprite.eo.tools`
Python entry point group.
